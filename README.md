# Guna Sundari S - Portfolio Website

A professional, responsive portfolio website for Guna Sundari S, Business Transformation Associate.

## How to View
Simply open `index.html` in any web browser. No server or installation is required.

## Project Structure
- `index.html`: Main HTML content containing all sections (Hero, About, Skills, Experience, Projects, Contact).
- `style.css`: All visual styles, variables, and responsive design rules.
- `script.js`: Simple JavaScript for smooth scrolling navigation and button interactions.

## Customization
- **Resume**: Replace `resume.pdf` with your actual file.
- **Links**: Update `href="#"` in `index.html` with your actual LinkedIn or Email links.
- **Images**: If you wish to add images, place them in an `images/` folder and update the `src` attributes.

## Deployment
This is a static site and can be deployed easily to:
- GitHub Pages
- Netlify (Drag and drop the folder)
- Vercel
